# 임베디드 시스템의 설계
# 습관 형성 스마트 잠금 의자 ( 집중 의자 )

## 1. 문제 인식
학생들중에는 간혹 공부 습관이 잡혀있지않아 1시간동안 집중은 커녕 앉아있는것도 힘든 학생들도 있습니다.
이들의 자기 통제력 부족은 의지 뿐만 아니라 환경적 요인과 습관 형성의 부재에서 비록된다 판단하였습니다.
#### “앉아있기가 힘든 사람에게, 억지로라도 ‘앉아 있게 만드는 환경’을 주면 어떨까?”
라는 생각에 도달하게 되어 "습관 형성 스마트 잠금 의자"를 떠올렸습니다. 수업시간에 배운 임베디드 시스템을 이용하면 충분히 구현 가능할거라는 생각이였습니다.

## 2. 작품 개요
#### 사용자가 의자에 앉으면 안장(시트) 아래의 하중 감지 센서가 20kg 이상을 감지함. (제품마다 무게 상이)
공부 습관 형성이 중요한 나이 10세부터 사용이 가능하도록 제품 크기군을 다양하게 만들어 무게를 감지합니다.

#### 일정 하중이 감지되면, 팔걸이 부분의 자동 잠금장치가 작동하여 사용자의 **허벅지 위로 바(bar)**가 내려옴.
바는 내려오면서 서로 맞물리는 구조로 사용자가 앉았을때 허벅지 위에 고정됩니다. 따라서 사용자는 간단한 스트레칭 자세는 취할 수 있지만 일어서지는 못합니다.

#### 사용자는 최소 50분간 고정 상태로 앉게 되며, 그동안 자리를 떠날 수 없음.
잠금시간은 앱으로 설정이 가능하지만 최소 잠금시간은 50분, 최대 잠금시간은 120분입니다. 또한 하루에 3번까지 긴급 잠금 해제를 할 수 있으며 팔걸이 옆면의 버튼을 통해 잠금을 해제할 수 있습니다

#### 설정 시간 경과 후 잠금장치가 자동 해제되어 자유롭게 이동 가능.
만약 잠금시간이 지났는데도 본인이 더 공부를 하고 싶을 수도 있으니 잠금 시간이 지났는데도 무게가 30초 이상 변화가 없으면 안장의 햅틱 반응을 통해 사용자에게 알리고 옆면의 버튼이 눌리기 전까지 잠금이 풀리지않습니다.

#### 잠금 시간은 앱 또는 블루투스로 조절 가능하게 확장 가능.
블루투스로 앱과 연결할 수 있으면 앱에는 무게 변화에 따른 집중 그래프, 잠금 시간 늘리기 응원 효과, 친구와 집중 배틀 등 다양한 효과를 적용할 수 있습니다

## 3. 주요 기능
### 자동 무게 감지 잠금 시스템
안장에 내장된 하중 센서(Load Cell) 가 30kg 이상의 무게를 감지하면 사용자가 앉았다고 인식.
인식 즉시 팔걸이에 부착된 잠금바가 아래로 내려오며, 허벅지 위쪽 공간(직접 닿지는 않음)을 가로질러 고정되어 사용자가 완전히 일어서지 못하게 함. 
자유로운 상체 움직임이나 간단한 스트레칭은 가능함.

### 시간 기반 자동 잠금 해제

설정된 잠금 시간이 지나면 팔걸이의 잠금장치가 자동 해제 대기 모드로 전환됨.

즉시 해제되지 않고, 안장 진동(햅틱 반응) 으로 ‘잠금 종료 알림’을 전달.

사용자가 원할 때, 팔걸이 왼쪽 세로 벽면의 해제 버튼을 눌러 직접 해제 가능.

### 앱 연동 기능 (Bluetooth 통신 기반)

##### 아두이노 + HC-06 블루투스 모듈을 이용해 모바일 앱과 연동.
#### 앱을 통해 다음 항목 설정 가능:

잠금 시간 설정: 최소 30분 ~ 최대 2시간까지 조정 가능.

긴급 잠금 해제: 하루 3회까지 긴급 해제 가능 (23:59:59 기준으로 자동 초기화).

사용 기록 조회: 하루 동안 집중한 시간, 해제 횟수 등 데이터 확인 가능

경쟁 시스템: 사용자를 익명처리하고 의자 사용자들과 하루 04:00에 초기화되는 랭킹 시스템. 순위에 상관없이 하루 7시간을 넘기면 아바타를 구매할 수 있는 토큰 지급

디자인적 요소: 앱 '열품타'와 같이 집중할때의 아바타를 설정가능하며 아바타는 인앱 구매 요소또는 경쟁 시스템을 통해 획득가능

인앱 구매 요소: 아바타를 구매할 수 있는 토큰을 구매하는 기능.

### 안전 및 편의 고려

긴급 상황(예: 낙상, 불편함 등) 시 버튼으로 즉시 수동 해제 가능.

잠금바는 부드러운 곡선 구조와 완충 패드로 제작하여 안전 확보.

전원은 USB 또는 C타입 포트로 공급 가능.
